## Social Media Post Image

**Task**: Create an eye-catching social media image for a post about [TOPIC]

**Requirements**:
- Dimensions: 1200x630 (optimal for most platforms)
- Include brand colors and typography
- Add engaging visual elements
- Clear call-to-action

**Prompt Template**:
```
Create a professional social media image for a post about [TOPIC]. Use a clean, modern design with:

- Background: [GRADIENT/SOLID COLOR/IMAGE]
- Main headline: "[HEADLINE TEXT]" in [FONT STYLE] at [SIZE]pt
- Subtext: "[SUBTEXT]" for additional context
- Visual element: [ICON/ILLUSTRATION/DIAGRAM] positioned [LOCATION]
- Brand colors: [PRIMARY COLOR], [SECONDARY COLOR]
- Call-to-action: "[CTA TEXT]" in [STYLE]

Make it visually appealing and optimized for social media sharing.
```

## Product Showcase Video

**Task**: Create a product demonstration video

**Requirements**:
- 1920x1080 resolution at 30fps
- 30-60 second duration
- Product shots with zoom effects
- Background music and voiceover
- Professional transitions

**Prompt Template**:
```
Create a professional product showcase video featuring [PRODUCT NAME]. Structure it as:

1. **Opening Clip** (5s): Title screen with product name and tagline
   - Background: [GRADIENT/TEXTURE]
   - Title animation: [STYLE] reveal
   - Subtitle: [TAGLINE]

2. **Product Shots** (20s): Multiple angles with Ken Burns effects
   - Clip 1: Wide shot with slow zoom in
   - Clip 2: Detail shot with pan across features
   - Clip 3: Usage demonstration
   - Transitions: Smooth fades between clips

3. **Features Section** (15s): Text overlays highlighting key features
   - Feature 1: [FEATURE] with icon
   - Feature 2: [FEATURE] with icon
   - Feature 3: [FEATURE] with icon

4. **Call-to-Action** (5s): Contact information and purchase prompt
   - Text: [CTA MESSAGE]
   - Background: [STYLE]

Audio: [BACKGROUND MUSIC STYLE] at 70% volume, [VOICEOVER SCRIPT] at 90% volume
```

## Explainer Video Script

**Task**: Create a short explainer video script with visuals

**Requirements**:
- 60-90 second video
- Clear narrative structure
- Visual metaphors
- Professional audio

**Prompt Template**:
```
Create an explainer video about [TOPIC] with this structure:

**Script Outline**:
1. **Hook** (10s): Attention-grabbing opening
   - Visual: [DRAMATIC SCENE/QUESTION ON SCREEN]
   - Audio: [UPBEAT MUSIC/VOICEOVER HOOK]

2. **Problem** (15s): Establish the problem or need
   - Visual: [PROBLEM ILLUSTRATION]
   - Voiceover: "Have you ever [PROBLEM STATEMENT]?"

3. **Solution** (20s): Introduce the solution
   - Visual: [PRODUCT/SERVICE DEMO]
   - Voiceover: "Introducing [SOLUTION NAME]"

4. **Benefits** (15s): Highlight key benefits
   - Visual: [BENEFIT ICONS/ANIMATIONS]
   - Voiceover: "With [SOLUTION], you get [BENEFITS]"

5. **Call-to-Action** (10s): Clear next steps
   - Visual: [CONTACT INFO/SIGNUP PROMPT]
   - Voiceover: "[CTA MESSAGE]"

**Video Specifications**:
- Resolution: 1920x1080
- Frame rate: 30fps
- Style: [ANIMATION STYLE: 2D/3D/LIVE ACTION]
- Music: [GENRE] background track
- Voiceover: Professional, confident tone
```
